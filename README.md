# What is this?
A simple CLI todo app that is POSIX-compliant.

# Target features
- [x] Add tasks
- [x] List tasks (with filters, tags, etc.)
- [x] Delete tasks
- [x] Mark tasks as completed
- [x] Edit tasks
- [x] Search tasks
- [x] Tag tasks
- [x] List tags
- [ ] Notifications

# Storage
- [x] SQLite
